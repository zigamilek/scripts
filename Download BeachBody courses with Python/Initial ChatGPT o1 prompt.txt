Please help me write a python script that will scrape videos and other materials from the beachbodyondemand.com website using selenium, selenium wire and yt-dlp.

Use Firefox as the browser, not headless.

The script will be provided a file with urls of individual programs to visit as a command line parameter. The second command line parameter will be the main output folder. You will create subfolders for each program.

You will have to "guess" the name of the program (which will also be the name of the subfolder) from the url and the <title>.

For example, if the url is https://www.beachbodyondemand.com/programs/live-it-up/start-here?locale=en_US
and the <title> is <title>Live It Up - High-Energy Workout with Shaun T</title>
then the name of the program and the subfolder is "Live It Up".
