please write a script that will open a JSON file and use yt-dlp to download videos from the urls provided in the file.

the script should use the cookies.txt file next to the script to pass it to yt-dlp as the --cookies parameter. it should also pass --external-downloader aria2c to yt-dlp.

the script should also take --output-folder as the command line parameter
